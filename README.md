# Course-Project

由 MongoDB、Express、React、Node.js 所構成的**MERN project**。(有RWD)

**簡介：**

- Server端
  - **註冊功能** (role: instructor、student)
  - **登入功能** 
  - **註冊課程功能** (帳號身分為instructor者專用)
  - **訂閱課程功能** (帳號身分為student者專用)
  
- Client端
  - **Axios對後端API發送Request**
    (註冊、登入、對DB的課程表單CRUD、顯示課程表單的課程資料)
